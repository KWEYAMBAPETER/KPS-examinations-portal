 Troubleshooting
Common Issues
Login Problems
Issue: Unable to login with correct credentials

Solution: Clear browser cache and refresh page

File Upload Issues
Issue: PDF files not uploading

Solution: Ensure file is PDF format and under size limit

Timer Not Working
Issue: Countdown timer stops or displays incorrectly

Solution: Refresh page and restart examination

Certificate Not Displaying
Issue: Certificate tab shows no certificate

Solution: Check if assessment is complete and score meets passing criteria
